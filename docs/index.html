Features

Nothing you don't.

Lightweight, privacy-focused, and designed for how you actually use the web. No subscriptions. No telemetry.

Instant picker panel

Click any link from Mail, Slack, Notes, or anywhere — a floating panel appears in the center of your screen listing every installed browser. Dismiss with Esc, pick with a click or number key.

Appears instantly on any link
Keyboard-first

Press 1–9 to instantly open in a specific browser. Use ↑↓ to navigate, Enter to confirm, Esc to cancel. Never touch the mouse if you don't want to.

Set a default, skip the dialog

Star any browser to make it your default. Links will open directly in it — no picker shown. Switch your default anytime from the menu bar in two clicks.

Zero friction when you need it
Private by design

No analytics, no network calls, no crash reporting. Every URL you click stays on your machine. The app lives quietly in your menu bar and only wakes up when a link is clicked.

How it works

Set up in
two minutes.

1

Set Browser Picker as your default browser

Open System Settings → Desktop & Dock → Default web browser and choose Browser Picker. This tells macOS to route every link click through it.

2

Click any link, anywhere

A floating panel appears in the center of your screen showing all your installed browsers. The URL you're about to open is shown at the top so you always know where you're going.

3

Pick a browser or set a default

Click, hover, or press a number key to open the link. Or star a browser in Settings to skip the picker entirely — useful if you mostly use one browser but occasionally want others.

⚠️ macOS will block the app on first launch — here's how to open it

Because Browser Picker isn't from the App Store, macOS shows a security dialog. You may see "BrowserPicker.app Not Opened". Follow these steps:

1
Click Done to dismiss the initial dialog.
2
Open System Settings → Privacy & Security.
3
Scroll to the Security section and click Open Anyway next to the BrowserPicker entry.
"BrowserPicker was blocked from use because it is not from an identified developer."
4
Enter your password or use Touch ID. You only need to do this once.
1
Unzip and move BrowserPicker.app to your /Applications folder.
2
Launch the app. If macOS blocks it, go to System Settings → Privacy & Security and click Open Anyway. See step-by-step guide ↑
3
Open System Settings → Desktop & Dock → Default web browser and choose Browser Picker.
4
Click any link — the picker panel appears. A globe icon in your menu bar means it's running.
